En este repositorio quiero que conozcan mi trabajo en este ejercicio usando Programación Orientada a Objetos (POO), Arreglos (arrays) e Interfaz gráfica con JFrame (Swing).
------------------------------
## Ejercicio Padlet
Este proyecto contiene la resolución del ejercicio programado en Java. La clase de inicio se encuentra dentro del paquete de ejecución principal.
## 📋 Requisitos Previos
Para ejecutar este proyecto de forma correcta, necesitas tener instalado:

* Apache NetBeans IDE (Versión 12 o superior recomendada).
* Java Development Kit (JDK) (Versión 8 o superior).

## 🚀 Instrucciones de Ejecución
Sigue estos pasos para abrir y correr la aplicación en tu entorno local:

   1. Clonar o descargar este repositorio en tu computadora.
   2. Abrir Apache NetBeans IDE.
   3. Ir al menú superior: File > Open Project.
   4. Buscar y seleccionar la carpeta raíz llamada EjercicioPadlet.
   5. Esperar a que el IDE cargue los módulos y dependencias del proyecto.
   6. En el panel izquierdo (Projects), expandir la siguiente ruta de carpetas:
   * Source Packages
      * program
   7. Hacer clic derecho sobre el archivo Principal.java.
   8. Seleccionar la opción Run File (o presionar Shift + F6).

## 📦 Estructura del Proyecto
La sección clave del código se organiza de la siguiente manera:

EjercicioPadlet/
